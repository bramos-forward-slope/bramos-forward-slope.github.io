## Angular JS

### Zebra stripe table

```
<table cellspacing="0" rules="all" border="1" style="border-collapse: collapse;">
    <tbody>
        <tr>
            <th>Select All</th>
            <th>Column A</th>
            <th>Column B</th>
            <th>Column C</th>
        </tr>
        <tr ng-repeat="r in vm.rows" ng-class="$index % 2 == 0 && 'gridAltRow' || 'gridRow'">
            <td><input type="checkbox" ng-model="vm.select[$index]" /></td>
            <td>{{r.columnA}}</td>
            <td>{{r.columnB}}</td>
            <td>{{r.columnC}}</td>
        </tr>
    </tbody>
</table>
```
